# Требования

- OpenJDK 14
- PostgreSQL 11
- Artemis или ActiveMQ

# Стек технологий

- Java 14
- JDBC
- JPA 2
- JAX-RS
- JMS
- Spring Boot 2.1
- Spring Cloud Greenwich
- Liquibase 3.6.2
- N2O Platform 4

# Структура проекта

- `rdm-sync-spring-boot-starter` - стартер для синхронизации с RDM.
- `rdm-sync-service` - запускаемый модуль для синхронизации с RDM на основе стартера.
